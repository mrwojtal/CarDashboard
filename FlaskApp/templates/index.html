Mini R50 Web Panel

Introduction

Website created to enable controlling LCD display on Mini R50 cluster.

Manual

User can control currently displayed values on LCD display after entering to control page, by clicking Control button. On the control page, user can choose one of five parameters to be displayed on the LCD displayed:

  1. Trip - distance with MIL indicator ON.
  2. Speed - current speed of the vehicle.
  3. Temperaure - current intake air temperature
  4. Range - expected range, based on the fuel consumption, calculated based on maximum fuel tank capacity.
  5. Consumption - current vehicle fuel consumption.

OBD Patameters table contains OBD responses to certain PID's. User can read current values of various parameters.
